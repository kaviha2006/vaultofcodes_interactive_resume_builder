## About the Project

This project was developed as part of the Vault of Codes assignment to demonstrate practical skills in frontend web development.

The Interactive Resume Builder enables users to:

Enter personal details through a form

Dynamically add multiple education and experience entries

Select skills interactively

View a live resume preview alongside the form

Experience smooth animations and transitions

Use the application seamlessly across desktop, tablet, and mobile devices

The focus of this project is on real-time updates, clean UI design, responsiveness, and user experience.

---

## Key Features

Live resume preview while typing

Dynamic addition of education and experience sections

Interactive skill selection

Smooth CSS animations and transitions

Responsive design using media queries

Clear form/reset functionality
